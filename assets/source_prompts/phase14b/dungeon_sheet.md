# Phase 14B Dungeon Sheet Prompt

Use case: stylized-concept
Asset type: pixel game dungeon sprite sheet for Git Dungeon
Primary request: create a compact pixel-art sprite sheet containing 15 separate dungeon assets in a clean 5 columns x 3 rows grid, no text labels.
Assets in row order: stone floor tile, stone wall tile, narrow corridor tile, open wooden dungeon door, locked iron dungeon door, closed treasure chest, open treasure chest, armed spike trap, spent dull spike trap, iron key, locked vault door, open vault door, current room marker, available room marker, boss gate.
Style/medium: crisp 16-bit pixel art, nearest-neighbor look, readable at tiny game scale, orthographic top-down RPG tiles and icons, not a UI mockup.
Composition/framing: each asset centered in its own square cell with generous padding, consistent scale, separated by clear empty spacing, no labels, no letters, no numbers.
Color palette: controlled dungeon palette with slate stone, muted moss green, warm gold highlights, red danger accents; avoid one-note purple or blue palette.
Constraints: perfectly flat solid #00ff00 chroma-key background across the entire sheet; no shadows, no gradients, no texture in the background; do not use #00ff00 inside any asset; no watermark; no text; do not copy Stardew Valley assets or style directly.
Avoid: smooth vector edges, painterly blur, photorealism, isometric perspective, text labels, UI frames.
