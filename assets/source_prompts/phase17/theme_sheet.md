# Phase 17 Theme Sprite Sheet Prompt

Use case: stylized-concept
Asset type: pixel art sprite sheet for a Python roguelike game, to be postprocessed into small transparent sprites
Primary request: Create an 8-cell sprite sheet for Git Dungeon Phase 17 theme assets.
Canvas and grid: one image, 4 columns x 2 rows, exactly eight separate cells. Each cell contains one centered pixel-art object with generous padding. Use a perfectly flat solid #ff00ff chroma-key background across every cell, no shadows, no gradients, no texture, no floor plane, no labels, no watermark, no text. Do not use #ff00ff or magenta anywhere in the objects.
Style: crisp 16-bit pixel art, readable at 16x16 to 96x32, limited palette, strong silhouette, warm torch amber plus git-green/cyan accents, dark stone dungeon technology-fantasy mood.
Objects by cell, left to right, top row then bottom row:
1. title_banner: a wide Git Dungeon banner object, stone-and-metal plaque with tiny branch-line motif, no readable letters.
2. dungeon_entrance: arched dungeon doorway shaped like a git branch gateway, glowing amber inside.
3. commit_shard: small crystal shard with subtle commit-node sparkle, cyan/green.
4. branch_door: small open door with branching path symbol suggested by pixels, no text.
5. merge_conflict_trap: dangerous floor trap with split red/green conflict lines and spikes.
6. ci_sentinel: compact mechanical sentinel eye/tower with green status light.
7. release_gate: sturdy final gate with gold seal and green gem, boss-ready silhouette.
8. torch_lit: small wall torch with warm flame for idle animation.
Avoid: readable words, UI text, logos, brand marks, smooth vector style, realistic painting, isometric perspective, blurry edges, anti-aliased softness, complex background.
